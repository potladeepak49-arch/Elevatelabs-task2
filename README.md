
#Task Todo

Task todo is a minimal and clean To-Do List web application developed by me using HTML, CSS, and Vanilla JavaScript.  
This project was built to practice front-end fundamentals and DOM manipulation.

# Features

- Add new tasks using the "+" button
- Add tasks by pressing the Enter key
- Mark tasks as complete using a checkbox
- Delete tasks using the cross (Ã—) button
- Dynamic task creation using JavaScript
- Instant UI updates without page reload


# Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

# Functionality

- The application uses JavaScript event listeners to handle user interactions.
- Tasks are dynamically created using DOM manipulation.
- The completed state is managed using class toggling.
- Tasks can be removed instantly from the UI without refreshing the page.
